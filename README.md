# TurtleMarket - 龟市

Turtle WoW 硬核模式 P2P 去中心化交易公告板插件。

硬核模式没有拍卖行，玩家只能面对面交易。龟市通过隐藏聊天频道实现商品发布、求购需求广播，帮助买卖双方快速匹配，然后通过游戏内密语联系、面对面完成交易。

## 安装

1. 下载 ZIP 并解压
2. 将文件夹重命名为 `TurtleMarket`（去掉 GitHub 自带的 `-main` 后缀）
3. 放入 `Interface/AddOns/` 目录，重启游戏
4. 输入 `/tm` 打开主面板

> 仅支持 Turtle WoW 服务器（Nordanaar / Tel'Abim / Turtle WoW）

## 界面总览

龟市主界面有 **5 个标签页**：

| 标签页 | 用途 |
|--------|------|
| **浏览** | 统一市场大厅，查看所有出售和求购信息 |
| **出售** | 从背包选择物品，设定价格，发布卖单 |
| **求购** | 输入物品名称和预算，发布买单 |
| **我的商品** | 管理自己发布的出售和求购，可取消或重新发布 |
| **历史** | 查看交易历史记录 |

## 浏览 - 统一市场大厅

浏览页同时展示所有玩家的 **出售商品** 和 **求购需求**：

- 每条记录前有彩色类型标签：绿色 **[出售]** / 橙色 **[求购]**
- **搜索**：输入物品名称关键词搜索
- **类型筛选**：点击「全部 / 出售 / 求购」按钮快速过滤
- **价格筛选**：设置最低价和最高价范围
- **玩家筛选**：按玩家名过滤
- **只看在线**：仅显示当前在线的玩家
- **排序**：按价格升/降序、时间、数量排序
- **密语**：选中一条记录后，点击底部密语按钮直接联系对方
  - 选中出售商品 → 按钮显示「密语卖家」，发送购买意向
  - 选中求购信息 → 按钮显示「密语买家」，告知你有货
- **Tooltip**：鼠标悬停查看物品完整属性、卖家/买家信誉

## 出售 - 发布卖单

1. 切换到「出售」标签页
2. **打开背包**，点击你想出售的物品（物品信息会自动填入预览框）
3. 填写 **数量** 和 **价格**（金/银/铜）
4. 可选填写 **备注**（如在线时间、小号名）
5. 点击「发布商品」

发布成功后，你的商品会出现在所有安装了龟市的玩家的浏览页中。

## 求购 - 发布买单

1. 切换到「求购」标签页
2. 选择物品（三种方式任选）：
   - **点击背包物品**：直接点击背包中的物品，自动填入名称和图标
   - **点击聊天链接**：点击聊天窗口中的物品链接，自动识别
   - **手动输入**：在输入框中直接输入物品名称
3. 填写 **数量** 和 **预算上限**（金/银/铜）
4. 可选填写 **备注**
5. 点击「发布求购」

发布后，你的求购信息会出现在浏览页中，卖家可以看到并主动联系你。

## 我的商品

- 上半部分显示你发布的 **出售商品**，下半部分显示你的 **求购信息**
- 选中后可以 **取消** 出售或求购
- 已过期的商品可以一键 **重新发布**（自动复制原价格、数量和备注）

## 信誉系统

每次交易完成后会弹出评价窗口（好评 / 中评 / 差评，10 秒不点默认好评）。

信誉分为四级，显示在玩家名旁：

| 等级 | 颜色 | 条件 |
|------|------|------|
| 新手 | 灰色 | 初始 |
| 可靠 | 绿色 | 多次好评 |
| 信赖 | 蓝色 | 大量好评 |
| 元老 | 金色 | 长期优质交易 |

## 交易辅助

- 与玩家打开交易窗口时，自动显示该商品的预期价格
- 对方给的金额不匹配时会出现 **红色警告**
- 交易完成后自动下架商品、记录到历史
- 收到交易密语时播放 **音效提示**

## 配置

点击主面板右上角的齿轮按钮进入设置：

- **过期时间**：商品自动下架时间（24/48/72/168 小时）
- **缓存上限**：本地缓存的商品数量（100-2000）
- **心跳间隔**：在线状态广播频率
- **密语格式**：中文 / English
- **音效开关**：密语提示音效
- **交易评价**：是否在交易后弹出评价窗口

## 常用命令

| 命令 | 作用 |
|------|------|
| `/tm` 或 `/market` | 打开/关闭主面板 |
| `/tm post` | 直接打开出售页 |
| `/tm my` | 直接打开我的商品 |

## 注意事项

- 龟市是 **P2P 公告板**，不是拍卖行。所有通信通过游戏内聊天频道传输，仅安装了龟市的玩家之间可见
- 插件不会自动完成交易，买卖双方需要通过 **密语联系** 后 **面对面交易**
- 商品发布后有有效期，过期会自动下架。可以在「我的商品」中重新发布
- 在线状态依赖心跳广播，下线后约 10 分钟显示为离线
- 如果浏览页为空，可能是附近没有其他龟市用户在线，耐心等待同步即可
