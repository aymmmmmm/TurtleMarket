# TurtleMarket - 龟市

Turtle WoW 硬核模式 P2P 去中心化交易公告板插件。

硬核模式没有拍卖行，玩家只能面对面交易。龟市通过隐藏聊天频道实现商品发布、求购需求广播，帮助买卖双方快速匹配，然后通过游戏内密语联系、面对面完成交易。

## 功能

### 出售与浏览
- 从背包选择物品，设定价格，一键发布到全服
- 浏览所有在售商品，支持关键词搜索、价格范围筛选、卖家过滤、只看在线
- 按价格/时间/数量排序，分页浏览
- 物品图标真实显示（通过协议传输纹理路径，三级回退机制）
- 鼠标悬停显示原生物品 Tooltip（装绑、属性、套装等完整信息），并附加卖家/信誉/价格
- 点击「密语卖家」直接发送格式化的购买意向密语

### 求购
- 手动输入物品名或 Shift+点击物品链接发布求购（链接方式自动识别图标和 ID）
- 卖家可在求购列表中找到买家需求，点击「密语买家」主动联系
- 求购支持搜索、排序、分页

### 信誉评分
- 四级信誉：新手(灰) → 可靠(绿) → 信赖(蓝) → 元老(金)
- 交易完成后弹出评价窗口：好评 / 中评 / 差评（10秒不点默认好评）
- 信誉等级显示在浏览列表和求购列表的玩家名旁

### 我的商品
- 查看自己发布的出售和求购列表
- 取消出售 / 取消求购
- 过期商品一键重新发布（复制原价格和数量，生成新有效期）

### 交易辅助
- 交易窗口旁自动显示预期价格提示
- 金额不匹配时红色警告
- 交易完成后自动下架、记录历史
- 收到交易密语时播放音效提示

### 配置面板
- Tab 栏当前页签金色高亮，快速区分活跃标签
- 默认过期时间（24/48/72/168小时）
- 最大缓存数（100-2000）
- 心跳间隔（在线状态广播频率）
- 交易评价开关、密语格式（中/英）、音效开关

## 使用方法

```
/tm        打开/关闭主面板
/tm post   打开出售界面
/tm my     打开我的商品
/tm debug  切换调试模式
/market    同 /tm
```

### 通信协议

所有消息通过隐藏的 `TurtleMarket` 聊天频道传输：

| 消息 | 用途 | 格式 |
|------|------|------|
| `#P` | 发布商品 | `id:itemId:name:count:gold:silver:copper:hours:seller:time:texture` |
| `#C` | 取消商品 | `listingId:sellerName` |
| `#W` | 发布求购 | `itemId:name:count:maxGold:maxSilver:maxCopper:buyer:time` |
| `#X` | 取消求购 | `wantId:buyerName` |
| `#H` | 心跳 | `playerName:listingCount:timestamp` |
| `#S` | 同步请求 | `digest` |
| `#D` | 同步数据 | `entry1;entry2;...` |
| `#F` | 分片消息 | `msgId\|partNum\|totalParts\|data` |

### 数据持久化

通过 WoW SavedVariables 机制自动保存：

- `TM_Data` — 商品列表、求购列表、我的挂单、交易历史、配置、信誉数据
- `TM_PlayerCache` — 玩家名→职业映射缓存

### 关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 消息长度上限 | 250 字符 | WoW 频道消息限制，超长自动分片 |
| 发送节流 | 1秒/条，突发3条，5秒冷却 | 防止频道刷屏 |
| 分片上限 | 100 片/消息 | 防止超大消息 |
| 心跳间隔 | 300 秒 | 可配置 |
| 在线判定 | 600 秒内有心跳 | - |
| 缓存上限 | 500 条 | 超限淘汰最旧，可配置 100-2000 |
| 过期清理 | 到期 + 72小时无心跳 | 自动执行 |
| 历史记录 | 最近 100 条 | 环形缓冲区 |
| 每页条数 | 10 条 | - |

## 安装

将 `TurtleMarket` 文件夹放入 `Interface/AddOns/` 目录，重启游戏。

## 要求

- Turtle WoW 客户端 (Interface 11200)
- 仅在 Turtle WoW 服务器上运行（内置服务器检测：Nordanaar / Tel'Abim / Turtle WoW）
